## DoD 合作伙伴关系经济概要

| 组件 | 数值 |
|-----------|------:|
| 优先股投资 | $400M |
| 转换价格 | $30.03/股 |
| 发行认股权证 | 行权价 $30.03 |
| DoD 合计持股（完全转换 + 完全行权）| 约 15% 普通股 |
| NdPr 价格底 | **$110/kg**，10 年 |
| 价格底结算机制 | 如现货 < $110，每季度现金补差 |
| 五角大楼贷款（分离产能扩张）| $150M |
| 10X Facility 包销 | 磁铁产出的 100%，商业化后 10 年 |

**估值含义**: 如果 2027 年 NdPr 现货 = $85/kg（2020–2024 历史均值），
价格底提供 $25/kg × 6,000 MT = **每年约 $150M 现金补差**。这是一个由政府
担保的直接现金流流入，期限约 10 年。按 12% 折现率，现值约 $800M —— 内嵌
战略价值未在尾部倍数中充分反映。

## 估值敏感性

### 在不同 EV/EBITDA 倍数与 2027 年 EBITDA 情景下

| 2027 调整后 EBITDA | 15×（周期性）| 20×（特殊性）| 25×（战略性）| 30×（类公用事业）|
|----------------:|---------------:|----------------:|----------------:|-------------------:|
| $200M（空头情景）| $3.0B / $17 | $4.0B / $23 | $5.0B / $29 | $6.0B / $34 |
| $350M（基础情景）| $5.3B / $30 | $7.0B / $40 | $8.75B / $50 | $10.5B / $60 |
| $500M（良好情景）| $7.5B / $43 | $10.0B / $57 | $12.5B / $71 | $15.0B / $86 |
| $700M（多头情景）| $10.5B / $60 | $14.0B / $80 | $17.5B / $100 | $21.0B / $120 |

*（每股价格假设 DoD 优先股转换后完全摊薄股数为 175M；净债务调整在 $0–1B
级别下影响很小，因为现金余额充足）*

**当前 $60.18 隐含**: $350M EBITDA × 30× 或 $500M EBITDA × 20× 或
$700M EBITDA × 15×。都处于基础到多头情景的合理到可能区间内。
